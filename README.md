# Excel Link Cleaner

Локальный Streamlit-инструмент для проверки ссылок в Excel-файлах.

## Что умеет

- проверяет ссылки из выбранной колонки
- работает с одним листом или сразу со всеми листами
- проставляет статусы `ok`, `redirect`, `blocked`, `failed`, `invalid`, `empty`, `blacklist`
- может добавлять техническую колонку с причиной результата
- подсвечивает проблемные ссылки в итоговом файле
- использует `blacklist.txt`
- запоминает удачные варианты колонок и файлов в `known_names.json`

## Файлы проекта

- `check_links.py` — основная логика проверки и обработки workbook
- `app.py` — интерфейс на Streamlit
- `known_names.json` — локальная память по колонкам и файлам
- `blacklist.txt` — список доменов для мгновенного отсечения
- `settings.ini` — опциональные настройки
- `requirements.txt` — зависимости

## Запуск

Установка зависимостей:

```bash
pip install -r requirements.txt
```

Запуск приложения:

```bash
python -m streamlit run app.py
```

Или через bat-файл:

```bat
run_app.bat
```

## Как пользоваться

1. Загрузите `.xlsx` файл.
2. Выберите режим обработки листов.
3. Укажите, как выбирать колонку со ссылками: автоматически, по названию или по номеру.
4. При необходимости включите `tech`-колонку.
5. Настройте таймаут, количество потоков и blacklist.
6. Запустите проверку и скачайте итоговый файл.

## Примечания

- если сайт отвечает редиректом, ссылка попадет в `redirect`
- если материал выглядит удаленным или страница похожа на `not found`, результат будет `failed`
- если домен в blacklist, сетевая проверка не выполняется
- во время долгой проверки можно остановить процесс из интерфейса
